# Sprachprüfung der Projektdateien

**Kriterien:**

- **DQR 3 – Verständlichkeit:** Texte für Personen mit Berufsausbildungsniveau – also einfache, klare Sätze, keine Fachsprache ohne Erklärung, keine langen Schachtelsätze.
- **Anglizismen:** Englische Wörter, die durch ein deutsches Wort ersetzt werden könnten.

## 2. Wiederkehrende Probleme (dateiübergreifend)

| Begriff                             | Vorschlag                                | Kommentar                                                                               |
| ----------------------------------- | ---------------------------------------- | --------------------------------------------------------------------------------------- |
| **Onboarding**                      | Einarbeitung                             | In wf4 zentraler Begriff – dort konsequent ersetzen oder beim ersten Vorkommen erklären |
| **kollaborativ**                    | gemeinsam / miteinander                  | Fremdwort, das viele auf DQR 3 nicht kennen                                             |
| **hybrid**                          | vor Ort und online                       | Zentraler Begriff in WF3, aber nirgends erklärt                                         |
| **Lab** (Lab-Briefing, Lab-Debrief) | Übungsraum, Einführung, Auswertungsrunde | Fachjargon aus Fortbildungsbereich, für Kita-Teams ungewohnt                            |
| **Gallery Walk**                    | Rundgang, Ausstellungsrundgang           | Pädagogischer Fachbegriff, aber auf DQR 3 kaum bekannt                                  |
| **Cheat-Sheet**                     | Kurzübersicht, Schnellübersicht          | Leicht ersetzbar, kein Mehrwert des englischen Begriffs                                 |
| **Output**                          | Ergebnis                                 | Ohne Kontext unklar, einfach ersetzbar                                                  |
| **affin** (digital-affin)           | erfahren mit Technik / technikvertraut   | Auch Fremdwort, nicht auf Anhieb verständlich                                           |
| **SSO / Single Sign-On**            | einheitliche Anmeldung                   | Technischer Begriff, muss erklärt werden                                                |
| **remote**                          | von zu Hause / von unterwegs             | Nicht auf Anhieb klar für alle                                                          |
| **Timeline**                        | Zeitplan, Ablauf                         | Leicht ersetzbar                                                                        |
| **Briefing / Debrief**              | Einweisung / Auswertung                  | Nur im Moderationsplan, aber ersetzbar                                                  |
